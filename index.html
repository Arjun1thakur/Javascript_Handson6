Q7. Write a function that accepts parameter n and returns factorial of n)

Ans. 
                function factorial(n){
                    let no=1;
                    for(let i=n; i>=1; i--){
                      no*=i
                    }
                    return no;
                  }
                  console.log(factorial(0));
